# Contact Form Web Application

A simple Java Full Stack project built using HTML, CSS, Spring Boot, Spring Data JPA, and MySQL.

## Features
- Responsive Landing Page
- Contact Form (Name, Email, Message)
- Spring Boot Backend
- MySQL Database Integration
- Spring Data JPA
- REST API endpoint (/contacts)
- JSON Response

## Technologies Used
- Java
- Spring Boot
- Spring Data JPA
- MySQL
- HTML
- CSS
